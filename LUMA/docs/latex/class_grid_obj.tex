To allow a generalised application of the regularised BC we need to implement the following\+:
\begin{DoxyEnumerate}
\item Check normal directions to find orientation of normal wall;
\item Store the unknown and known directions in two vectors
\item Apply the boundary condition. Note\+: for the corners, the unknown distributions are too numerous to evaluate the density so must use extrapolation for the 4 corners (2D) or 8 corners (3D) from the bulk flow to find the density. Hence, we handle this case after the normal cases.

If it reaches here and has gone through all the directions then the wall orientation cannot be found based on the fact that it cannot find an adjacent site within the centre of the domain on which to base the BC. In this case it must be a buffered layer of inlet sites (the \char`\"{}second row\char`\"{} of sites you get when embedding a sub-\/grid in the inlet) so just set it to default values as it doesn\textquotesingle{}t affect the domain anyway.
